# Portfolio Rafeyfa Ashyla Putri

Website portfolio pribadi berbasis HTML, CSS, dan JavaScript ringan dengan tema merah muda (feminine), efek glassmorphism, animasi AOS, serta layout responsive.

## Fitur

- Hero section untuk perkenalan diri
- Tema visual merah muda dengan gradient dan glow
- Background video + overlay atmosferik
- Glassmorphism card pada container utama dan statistik
- Foto profil berbentuk lingkaran
- Tombol aksi "Hire me"
- Ikon sosial media menggunakan Boxicons
- Animasi elemen menggunakan AOS
- Mobile menu (hamburger) untuk layar kecil
- Responsive untuk desktop dan mobile

## Teknologi

- HTML5
- CSS3
- JavaScript (vanilla)
- AOS Animation (CDN)
- Boxicons (CDN)
- Google Fonts (CDN)

## Struktur Proyek


## Cara Menjalankan

Karena ini website statis, tidak perlu instalasi package.

1. Masuk ke folder proyek.
2. Buka `index.html` langsung di browser.

Opsional untuk development: gunakan Live Server (VS Code/WebStorm) agar auto-reload saat edit file.

## Kustomisasi Cepat

- Ubah nama, deskripsi, dan teks profil di `index.html`
- Ubah link sosial media pada daftar ikon di `index.html`
- Ganti foto profil di `images/zihad.webp`
- Ganti logo di `images/logo.webp`
- Ganti video background di folder `images/`
- Atur warna, tipografi, dan layout di `style.css`

## Deployment

Bisa di-deploy ke static hosting seperti:

- GitHub Pages
- Netlify
- Vercel

## Catatan

`index.html` menggunakan CDN untuk AOS, Boxicons, dan Google Fonts. Pastikan koneksi internet aktif agar ikon, font, dan animasi termuat sempurna.

## Lisensi

Proyek ini dibuat untuk kebutuhan portfolio pribadi dan pembelajaran.
